Bisa dipergunakan untuk memanage :
- Data pasien ibu hamil
- Data pasien bayi
- Data pasien ibu KB
- Data pasien umum

Dan menampilkan laporan :
- Rekam medis pasien
- Pemasukan per hari, minggu, bulan dan tahun

Program dibuat dengan mempergunakan stack :
- Apache web server
- PHP 7.x
- MySQL 5.x / MariaDB 10.x
